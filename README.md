# gaussian-splatting-submit

这个目录是从 `cgj-gaussian-splatting` 裁剪出来的纯代码提交副本，当前保留的是比赛主链能力，而不是原始上游仓库的完整说明文档。

## Render Test Views

```
python scripts/render_track_bundle.py \
  --bundle /path/to/bundle.ckpt \
  --output-dir /path/to/output
```

Please download the bundle checkpoints from the links below:
